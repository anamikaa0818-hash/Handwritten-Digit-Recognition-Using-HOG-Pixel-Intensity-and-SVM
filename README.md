# Model Evaluation - Handwritten Digit Recognition

This branch focuses on evaluating the performance of the trained machine learning model for handwritten digit recognition.

## Objective
To measure how accurately the model predicts handwritten digits using evaluation metrics.

## Contents
- 05_model_evaluation.ipynb → Notebook for evaluating model performance
- Visualizations and result analysis

## Evaluation Process
- Load trained model
- Make predictions on test dataset
- Compare predicted vs actual labels
- Calculate accuracy and performance metrics

## Technologies Used
- Python
- NumPy
- Scikit-learn
- Matplotlib
