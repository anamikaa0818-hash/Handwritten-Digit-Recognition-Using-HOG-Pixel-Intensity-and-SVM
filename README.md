# Model Training - Handwritten Digit Recognition

This branch focuses on training a machine learning model for handwritten digit recognition using feature extraction techniques.

## Objective
To build a classification model that can accurately recognize handwritten digits from image data.

## Contents
- 04_model_training.ipynb → Notebook for model training
- Training pipeline and model generation

## Training Workflow
- Load MNIST dataset
- Extract features using:
  - HOG (Histogram of Oriented Gradients)
  - Pixel intensity
- Train classification model (SVM)
- Evaluate basic performance during training
- Save trained model for later use
