# Data Exploration - Handwritten Digit Recognition

This branch focuses on exploring and understanding the dataset used for handwritten digit recognition.

## Objective
To analyze the dataset and gain insights before building machine learning models.

## Dataset
- Dataset: MNIST
- Contains images of handwritten digits (0–9)
- Each image: 28 × 28 pixels

## Tasks Performed
- Loaded dataset using sklearn
- Checked data shape and structure
- Visualized sample digit images
- Analyzed pixel intensity distribution
- Observed class distribution

## Key Insights
- Dataset contains balanced classes (digits 0–9)
- Pixel values range from 0 to 255
- Clear patterns exist in digit shapes
